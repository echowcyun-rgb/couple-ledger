# 2026-06-26 记账App项目初探

## 项目概览
- 情侣记账App（默认成员小五&小云），像素复古风UI，3套糖果色主题
- 技术栈：Next.js 16 + React 19 + TS + Tailwind v4 + Recharts + xlsx
- 数据存储：纯localStorage，无后端
- 通过v0.app生成

## 代码审查发现的问题
1. 导出xlsx假的（只toast不生成文件）
2. 导入只解析不写入数据
3. 记账提醒开关是摆设
4. 没有删除账单功能
5. 发薪日设置不能真正修改每个人的发薪日（只弹toast）
6. 首页只硬编码显示2个成员头像
7. 记账只能记当天，不能选过去日期
8. 复盘页缺年份显示
9. useLedger.ts 600行单hook
10. next.config ignoreBuildErrors
11. globals.css和ledger.css --accent变量冲突
12. MemberPage.tsx有重复prop setEditAvatar
13. 无PWA配置
14. 无数据同步/认证
